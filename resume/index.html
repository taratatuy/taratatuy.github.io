Сервер для учебного проетка
Бэкэнд на Node.js. Проект написан в качестве курсовой работы в вузе. API для передачи аудио файлов с сервера на фронт (запрос к сторонним API не делается, аудио просто берется из соответсвующей папки на сервере). Используется библиотеки: express.js (API), mongoose (работа с mongodb), dotenv (переменные окружения), jsonwebtoken (JWT), bcrypt (шифрование ключей), sentry/node (логи в облако).
